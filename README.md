# chess
Шахматы для двух пользователей. 
## Что умеют?
- Запись ходов
- Отображение текущего состояния игры: ожмдание хода, ход, шах или мат
- Невозможность подвести короля под удар при шахе
- Возможность сдаться, покинув игру

## Установка

npm i ws

npm i -d typescript

npm i

## Запуск

npm run build

npm start
